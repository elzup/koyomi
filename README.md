# koyomict

コンテンツの歴史をガントチャートで確認できるサイト。

## Features

- ガントチャート形式でコンテンツの年表を表示
- カテゴリ別フィルター・検索
- ドラッグ範囲選択 / ホイールズーム
- Google ログインでデータを Firestore に保存

## Tech Stack

- React 19 + TypeScript + Vite
- Firebase (Auth / Firestore / Hosting)

## Development

```bash
ni        # install
nr dev    # http://koyomict.localhost:1355
nr test   # vitest
nr build  # production build
```
